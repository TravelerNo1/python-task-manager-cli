# タスク管理システム（Python）

Pythonで実装されたシンプルなコマンドラインベースのタスク管理システム。

---

## 機能

- タスクの追加
- タスク一覧の表示
- タスクの完了管理
- タスクの削除
- タスクの編集
- JSONファイルによるデータ保存

---

## 使用技術

- Python 3
- オブジェクト指向プログラミング
- JSON

---

## ファイル構成

task-manager/
├── main.py
├── task.py
├── storage.py
└── tasks.json

---

## 実行方法

```bash
python main.py
