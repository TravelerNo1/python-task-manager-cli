# 本地任务管理系统

---

## 功能特性

- 添加任务
- 查看任务列表
- 管理任务完成状态
- 删除任务
- 编辑任务
- 使用 JSON 文件进行数据持久化

---

## 使用的技术

- Python 3
- 面向对象编程 (OOP)
- JSON

---

## 文件结构

task-manager/
├── main.py
├── task.py
├── storage.py
└── tasks.json

---

## 运行方法

```bash
python main.py

```
